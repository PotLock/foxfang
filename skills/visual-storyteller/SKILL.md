# Visual Storyteller

Expert visual communication specialist for creating compelling visual narratives, multimedia content briefs, and brand storytelling through design direction.

## When to Use This Skill

Activate when the task involves:
- Creating visual storytelling campaigns or brand narratives
- Developing storyboards, mood boards, or visual concept briefs
- Planning video content, animations, or motion graphics direction
- Designing infographics, data visualizations, or information design
- Cross-platform visual content adaptation (Instagram, TikTok, YouTube, LinkedIn, etc.)
- Photography or illustration art direction

## Visual Narrative Framework

Every visual story must follow a clear structure:

### Story Arc
1. **Setup** — Establish context, introduce the protagonist (often the customer/user)
2. **Conflict** — Present the problem or challenge driving the narrative
3. **Resolution** — Show how the brand/product provides the solution
4. **Call-to-Action** — Guide the audience to the next step

### Emotional Journey Mapping
- Map emotional peaks and valleys throughout the story
- Design visual pacing — rhythm and timing of visual elements
- Use color psychology and visual metaphors to reinforce emotions
- Build tension through progressive disclosure of information

## Multimedia Content Briefs

When creating content briefs, always include:

### Video / Animation Briefs
- **Format**: Aspect ratio, duration, platform target
- **Storyboard**: Scene-by-scene breakdown with visual descriptions
- **Shot List**: Camera angles, transitions, pacing notes
- **Audio Direction**: Music mood, voiceover tone, sound effects
- **Text Overlays**: Key messages, CTA placement, typography style

### Static Visual Briefs (Infographics, Social Posts, Ads)
- **Layout**: Grid structure, visual hierarchy, focal points
- **Color Palette**: Primary, secondary, accent — tied to brand
- **Typography**: Headlines, body, CTA font styles
- **Imagery**: Photography style, illustration approach, icon sets
- **Data Visualization**: Chart types, annotation style, legend placement

## Cross-Platform Adaptation Rules

| Platform | Format | Duration/Size | Key Considerations |
|----------|--------|---------------|-------------------|
| Instagram Stories | 9:16 vertical | 15s per slide | Interactive elements, swipe CTA |
| Instagram Feed | 1:1 or 4:5 | Single or carousel | First slide = hook, last = CTA |
| TikTok | 9:16 vertical | 15-60s | Trend-aware, native feel, text overlays |
| YouTube | 16:9 horizontal | 2-10min | Thumbnail optimization, first 5s hook |
| LinkedIn | 1:1 or landscape | Carousel PDF or video | Professional tone, data-driven |
| Pinterest | 2:3 vertical | Static or idea pin | SEO-optimized description, seasonal |
| Website | Responsive | Hero, section, modal | Interactive elements, scroll behavior |

## Information Design Principles

When transforming complex data into visuals:
- **Simplify first**: Remove noise, highlight the key insight
- **One idea per visual**: Don't overload a single frame
- **Use visual metaphors**: Familiar objects to represent abstract concepts
- **Progressive disclosure**: Layer information for scannability
- **Accessibility**: Minimum contrast ratio 4.5:1, alt text, colorblind-safe palettes
