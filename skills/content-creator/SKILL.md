# Content Creator

Expert content strategist and creator for multi-platform campaigns. Develops editorial calendars, creates compelling copy, manages brand storytelling, and optimizes content for engagement across all digital channels.

## When to Use This Skill

Activate when the task involves:
- Creating blog posts, articles, whitepapers, or case studies
- Developing editorial calendars and content pillars
- Writing video scripts, podcast outlines, or social media copy
- Brand storytelling and narrative development
- Content repurposing across platforms
- Content performance analysis and optimization

## Content Strategy Framework

### Content Pillars (define 3-5 per brand)
Each pillar should map to a business goal and audience need intersection. Check BRAND.md for brand voice and target audience before creating.

### Content Mix
- **Educational (40%)**: How-tos, guides, tutorials, industry insights
- **Entertaining (25%)**: Stories, behind-the-scenes, culture content
- **Inspirational (20%)**: Customer success stories, vision pieces, thought leadership
- **Promotional (15%)**: Product features, launches, offers — always value-wrapped

### Multi-Format Creation
| Format | Best For | Key Elements |
|--------|----------|--------------|
| Blog post | SEO, authority | 1500-2500 words, keyword-optimized, internal links |
| Video script | Engagement, demos | Hook in 3s, clear structure, CTA |
| Social copy | Awareness, traffic | Platform-native, hashtags, visual pairing |
| Email | Nurture, conversion | Subject line A/B, personalization, single CTA |
| Case study | Trust, sales enablement | Problem → Solution → Results with data |
| Infographic | Shareability, data | One key insight, scannable, branded |

## Content Creation Process

1. **Brief**: Define audience, goal, format, keywords, CTA, distribution channels
2. **Research**: Use `search_web` for trends, competitor content, data points
3. **Outline**: Structure with clear H2/H3 hierarchy, key messages per section
4. **Draft**: Write in brand voice (per BRAND.md), lead with value, end with CTA
5. **Optimize**: SEO elements, readability score, visual suggestions
6. **Deliver**: Save via `write_artifact` with proper filename

## Output Requirements

Always use `write_artifact` to produce deliverables:
- Blog posts → `blog_post.md`
- Video scripts → `video_script.md`
- Social media packs → `social_pack.md`
- Editorial calendars → `content_calendar.md`
- Email sequences → `email_sequence.md`
- Case studies → `case_study.md`

## Quality Checklist
- [ ] Aligned with brand voice (BRAND.md)
- [ ] Clear audience targeting
- [ ] Strong headline / hook
- [ ] Value before promotion
- [ ] Clear CTA
- [ ] SEO optimized (if applicable)
- [ ] Platform-appropriate format
- [ ] Saved via `write_artifact`
