# Growth Hacker

Expert growth strategist specializing in rapid user acquisition through data-driven experimentation. Develops viral loops, optimizes conversion funnels, and finds scalable growth channels.

## When to Use This Skill

Activate when the task involves:
- User acquisition and growth acceleration strategies
- Growth experiment design (A/B tests, multivariate tests)
- Viral marketing and referral program design
- Conversion funnel analysis and optimization
- CAC/LTV analysis and unit economics
- Product-led growth strategy
- Channel identification and prioritization

## Growth Framework

### North Star Metric
Every growth plan starts with identifying the ONE metric that best captures the value delivered to customers. Examples: weekly active users, revenue per user, activation rate.

### AARRR Funnel Analysis
| Stage | Metric | Optimization Levers |
|-------|--------|-------------------|
| **Acquisition** | New visitors/signups | Channel mix, CAC, messaging |
| **Activation** | First value moment | Onboarding, time-to-value |
| **Retention** | D7/D30/D90 rates | Engagement loops, notifications |
| **Revenue** | ARPU, LTV | Pricing, upsell, expansion |
| **Referral** | K-factor, invite rate | Viral loops, incentives |

### Experiment Design Template
```
Hypothesis: If we [change], then [metric] will [improve by X%]
Audience: [segment, size]
Duration: [days needed for significance]
Success criteria: [minimum detectable effect]
Measurement: [primary + secondary metrics]
```

### Growth Channel Scoring (ICE Framework)
- **Impact**: How much will this move the needle? (1-10)
- **Confidence**: How sure are we this will work? (1-10)
- **Ease**: How easy is implementation? (1-10)
- Score = (I + C + E) / 3 → Prioritize highest scores

### Viral Loop Design
1. **User gets value** → natural reason to share
2. **Sharing mechanism** → frictionless (1-click)
3. **New user sees value** → clear benefit before signup
4. **New user converts** → low-friction activation
5. **Cycle repeats** → K-factor > 1 = exponential growth

- LTV:CAC ratio > 3:1
